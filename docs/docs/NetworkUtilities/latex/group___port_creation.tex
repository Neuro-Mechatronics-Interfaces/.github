Initializes a UDP socket and configures server address. 

This function initializes a UDP socket and configures the server address using the provided parameters. It creates a UDP socket, sets it up with the specified server address and port, and assigns the socket to the provided pointer. If any errors occur during socket creation or address configuration, the appropriate error message is printed to stderr, Winsock resources are cleaned up, and the function returns EXIT\+\_\+\+FAILURE.
