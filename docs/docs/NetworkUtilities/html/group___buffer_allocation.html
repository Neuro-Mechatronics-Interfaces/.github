Detailed Description

Buffer byte size definitions.

All macros for byte allocation are defined using the following name syntax:
BYTES_<RECEIVING>_<PORT>_BUFFER
Where <RECEIVING> is the name of the receiving device, and
<PORT> is the name of the port for the corresponding UDP message socket.
